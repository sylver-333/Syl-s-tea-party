# ☕ Syl’s Tea Party

Welcome to **Syl’s Tea Party** – a whimsical, elegant tea-inspired website celebrating tea culture, cozy aesthetics, and boutique tea experiences.


Syl’s Tea Party is a modern boutique tea and coffee café located in NYC


This project is a simple front-end website designed to showcase a fictional tea shop featuring specialty teas, desserts, and a relaxing tea party atmosphere.

---

## 🌸 About The Project

**Syl’s Tea Party** is inspired by boutique tea salons and luxury tea experiences.  
The website highlights tea menus, cozy vibes, and elegant design.

Inspired by places like:
- Luxury tea salons
- Boutique cafés
- Afternoon tea culture

The goal of this project is to practice **HTML, CSS, and front-end design** while building a brand-style website.

---

## 🍵 Features

- Elegant tea shop landing page
- Menu section for teas and desserts
- Cozy aesthetic design
- Responsive layout
- Simple navigation
